BeForma Tester Interface

1) Start the backend from the project root:
   python -m uvicorn main:app --reload

2) Open this file in your browser:
   frontend/index.html

3) Default API base URL is:
   http://127.0.0.1:8000

You can test:
- AI Suggest Mode
- User Food Cart Mode
- Cart Preview
- Generate Diet + Workout Plan
- Workout Only
